# mac

### 快捷键
```
Command-Shift-4 			  截取所选屏幕区域到一个文件　　
Command-Shift-3 			  截取全部屏幕到文件　
Command-Q 				      关闭当前程序
Command-Shift-N 			  新建文件夹
Command-Delete 				  将文件移至废纸篓
Command-Shift-Delete 		清倒废纸篓
command+m 				      隐藏窗口
command+l 				      显示窗口
command+shift+. 			  显示所有以.开头的隐藏的文件
touch xxx  				      新建文件
command+shift+f				  全屏和退出全屏
command+control+f 			全屏和退出全屏
command+t 				      新建标签页
command+q 				      关闭程序
command+shift+t 			  快速打开上次关闭的页面
command+option+箭头 		切换到下一个标签页
command+w  				      关闭当前选项页
command+shift+b 			  打开关闭书签栏
command+option+b 			  打开书签管理器
command+y 				      打开历史记录
command+shift+j 			  打开下载内容页面
command+shift+delete		打开清空浏览数据对话框
ifconfig 				        查询ip地址
pwd     				        当前工作的路径
cd (不加参数）				    进root
cd ..					          进入上级目录
cd ~					          返回root
cd -					          返回上一个访问的目录
rm	xx				          删除文件名
cat					            在终端查看文件
ls（dir)					       文件目录
ls -ah					        查看隐藏的文件目录
cp 文件名  目标目录			将文件拷贝到目标目录下
mkdir					          新建文件夹
tar -zxf xxx.tar.gz			tar文件解压
unrar x abc.rar				  rar文件解压，需要安装rar工具
clear 					        清空当前输入
history					        查看历史输入记录
rm -rf xxxx             删除文件 
```
